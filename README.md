# Pokemon List By React-Virtualized

ポケモンの一覧をReactで描画してみました。ポケモンは約800匹もいるため、[react-virtualized](https://github.com/bvaughn/react-virtualized)を使って最適化をしています。

This app is inspired by [pokedex](pokedex).
